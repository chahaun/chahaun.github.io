# Node.js

## 패키지 매니저

### npm

npm은 Node Package Manager의 약자이며, 노드에서 자바스크립트 프로그램을 컴퓨터에서도  
실행할 수 있게 해주는데 대부분 자바스크립트 프로그램은 패키지 이름으로  
npm에 등록되어 있으므로 특정기능이 필요하면 npm에서 찾아 설치하면 된다.
